# Verse Roots

A Bible study app connecting English verses to their original Hebrew and Greek roots via Strong's concordance.

## Project Structure

- `apps/web/` — Vite + React + TypeScript frontend
- `data/ingestion/` — Node scripts that download STEPBible data and ingest it into SQLite
- `data/db/` — SQLite database file (gitignored, generated by ingestion)
- `data/raw/` — Downloaded raw TSV/text files (gitignored, cached locally)

## Re-running Ingestion

```bash
cd data/ingestion
npm install
npm run ingest   # downloads STEPBible files and populates data/db/verses.db
npm run verify   # queries Lam 3:22 and John 1:1 to confirm data integrity
```

Or from the workspace root:
```bash
npm run ingest
npm run verify
```

## Data Sources

All original-language data comes from [STEPBible-Data](https://github.com/STEPBible/STEPBible-Data),
licensed under [CC BY 4.0](LICENSES/STEPBIBLE-CC-BY-4.0.txt).

- **TAHOT** — Tagged Aligned Hebrew Old Testament (OSHB alignment)
- **TAGNT** — Tagged Aligned Greek New Testament (NA28 alignment)
- **TBESH** — Transliterated Brief lexicon of Extended Strongs for Hebrew
- **TBESG** — Transliterated Brief lexicon of Extended Strongs for Greek
